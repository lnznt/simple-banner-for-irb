= simple-banner-for-irb
== Description

バナーを表示するシンプルなスクリプトです。
単体実行もできますが、irb の起動時のバナー用に作りました。

下のようなバナーを表示します。
表示内容は、"RUBY"(バナー)、RUBY_DESCRIPTION、RUBY_COPYRIGHT です。


     ######  #     # ######  #     #
     #     # #     # #     #  #   #
     #     # #     # #     #   # #
     ######  #     # ######     #
     #   #   #     # #     #    #
     #    #  #     # #     #    #
     #     #  #####  ######     #



  ruby 1.9.2p0 (2010-08-18 revision 29036) [x86_64-linux]
  ruby - Copyright (C) 1993-2010 Yukihiro Matsumoto


"RUBY" と RUBY_DESCRIPTION の ruby バージョン部分は色付きで表示されます。

色は、ruby1.8 では「緑」、ruby1.9 では「赤」です。

また、"RUBY"はスクロールしながら表示されます。

== System Requirement

色付けやスクロールは TTY のエスケープシーケンスを使って制御しています。
エスケープシーケンスをサポートした TTY 上で動作します。

開発した環境は、Ubuntu Linux 11.04 です。環境変数 TERM の値は 'xterm' です。

  $ echo $TERM
  xterm

== Quick Start
~/.irbrc に以下を追加すると irb の起動時にスクリプトが実行されてバナーが表示されます。

  require 'banner'

.irbrc についての詳細は irb のマニュアルなどを参照してください。
